# AI Coach Bot for Virtual Influencer Projects

Telegram-бот на Python + aiogram, который помогает шаг за шагом собирать проект виртуального инфлюенсера:
идея → лор → внешность → локации → фотопак → видеопак → запуск.

## Что умеет MVP
- показывает план на сегодня
- ведёт по шагам
- принимает текстовые ответы
- даёт пример и упрощённое объяснение
- сохраняет прогресс
- поддерживает режимы easy / normal / sprint
- поддерживает ежедневную автo-отправку плана
- хранит настройки пользователя
- работает с SQLite

## Стек
- Python 3.13
- aiogram 3.x
- SQLAlchemy 2.x
- SQLite
- pydantic-settings
- long polling
